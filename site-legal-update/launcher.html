Несовместимые моды

Лаунчер находит сторонние моды в папке Mods и предупреждает о них — частая причина крашей и кика с серверов.

// КАК УСТАНОВИТЬ //

  1. 01

    Скачай DoomsdayLauncher.exe

    Кнопкой выше или прямой ссылкой /dl/DoomsdayLauncher.exe. Если основной домен не открывается, используй зеркало doomsday7dtd.ru или GitHub Release. Актуальная версия, размер и хэш публикуются в launcher.json.

  2. 02

    Запусти. Windows может показать предупреждение

    Проверь, что файл скачан с doomsday7dtd.online, doomsday7dtd.ru или GitHub Release, а SHA256 совпадает с launcher.json. Если есть сомнения — не запускай файл и напиши в Discord.

  3. 03

    Выбери способ установки

    «STEAM» — мод ставится прямо в твою Steam-папку. «КОПИЯ» — игра копируется в отдельную папку (для тех, кто хочет сохранить vanilla).

  4. 04

    Жми «ИГРАТЬ»

    Лаунчер сам отключит EAC, проверит мод и запустит игру с нужными ключами.

// ЧТО ДЕЛАТЬ ЕСЛИ //

«Игра не запускается / EAC kicked»
Открой лаунчер — он сам убьёт EAC-процессы. Если не помогло — проверь, что у тебя НЕТ другой папки 7 Days to Die, в которой EAC включён, и закрой её перед запуском.
«Мод не загружается / падает на старте»
Файлы мода повреждены. Открой лаунчер → нажми «ВОССТАНОВИТЬ МОД». Старые файлы удалятся и скачаются свежие с проверкой целостности.
«Не качается / зависает»
Прокси/VPN иногда мешают. Попробуй зеркало doomsday7dtd.ru или GitHub Release. Если всё ещё проблема — пиши в Discord.
«Antivirus ругается на лаунчер»
Скачай файл заново только с doomsday7dtd.online, doomsday7dtd.ru или GitHub Release и сверь SHA256 с launcher.json. Не добавляй exe в исключения вслепую; если предупреждение остаётся — напиши в Discord.
